## 支持漫画类型

- 支持一级，二级文件夹形式漫画
- 支持图片格式 webp, png, jpg, gif, bmp, apng, avif, svg
- 支持直接读取压缩包类型漫画, .tar, .tar.gz, .tar.bz2, .zip, .7z, .lzma, .cab

## 开发

```
yarn
yarn start
```

## 打包

```
yarn run package
```

### 开发说明

- 本软件可分为两种模 1. Electron 客户端模式 2. 在线模式。 在线模式适合转移手机端使用。

主要依赖

- @material-ui/core
- tailwindcss
- recoil
- react
- react-router
- electron-react-boilerplate
